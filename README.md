## Ürün Özellikleri

- Tür: Yarış, Hypercasual
- Platform: Mobile / Android
- Çeşitli güçlendirmelerle yarışta değişik avantajlar elde edilecek.
- Yarışmadaki AI rakiplerle yarışılacak.
- Map, statik yapıda olacak.
